## 一、JD 全文（DeepSeek Agent Harness 研发工程师）

> **团队使命**
> 我们正在把 DeepSeek 的前沿模型能力，转化为领先的 Agent 产品。这其中除模型本身以外的所有工作，都属于 Harness 的范畴。
>
> 你将加入 Harness 团队，与工程师、研究员、产品经理紧密协作，参与 DeepSeek 桌面端 Agent 产品研发的全过程，定义 DeepSeek 对 Harness 的理解。
>
> **主要职责**
> 1. 参与设计 DeepSeek 的 Harness 产品的技术架构与选型。
> 2. 参与开发 DeepSeek 的 Harness 产品。
> 3. 与研究员紧密合作，共同定义和实现 Harness 领域的前沿创新。
> 4. 与模型训练团队的工程师与研究员深度沟通与合作，参与实现模型与 Harness 的共同进化，从 Harness 的角度实现 DeepSeek 的 Harness 与模型的深度适配。

> 7. 协助项目管理相关工作。
>
> **任职要求**
> 1. 2 年以上软件开发从业经验，技术水平过硬，技术眼界广阔。特别优秀候选人可放宽年限。
> 2. 知名高校本科学历及以上。
> 3. 熟练使用 AI Agent 工具进行软件开发。在软件开发领域具有极强的学习能力。能够在 AI 辅助下，在没有直接经验的领域（如语言、技术、框架等）进行有质量保证的编程工作。
> 4. 是 Agent 产品的高强度用户，对 Agent Harness 的开发有极大的热情，对模型行为有品味有判断力，对开发者体验有强感知。深度使用过代码类及通用类 Agent 产品，并将相关产品的使用融入到自己的工作和生活中。
> 5. 熟悉 LLM 以及 Agent 基本机制及其技术原理，包括 LLM API、KV Cache、Agent Loop、Tool Use、Reasoning、Planning、Skills、MCP、Memory、Subagent、Multi-Agent 等相关知识。对 Prompt Engineering、Context Engineering、Harness Engineering 等课题有较深入的了解。
> 6. 良好的中文沟通能力。

> - AI 行业相关软件开发经验。
> - 作为小团队中的研发工程师快速迭代产品的经验。
> - 个人开源作品及开源社区的深度贡献。

> - 计算机科学领域、编程领域、或数据科学领域具有含金量的比赛奖项。

## 六、JD 里没写但一定会考的"暗线"

### 暗线 1：你能不能从 0 写出 minimal harness
JD 写"参与开发"——但小团队的工程师必须**能独立 ship 一个组件**。最直接的测试：让你白板写 minimal agent loop（30 分钟）。

**应对**：T-7 ~ T-2 把 [12 文档] 的 minimal harness 真的跑起来 + push 到 GitHub。

### 暗线 2：你和研究员能不能聊 paper
"前沿创新"在 JD 里出现两次。这意味着面试时会问你：
- "你最近读的 agent 相关 paper 是什么？"
- "你怎么看 [某篇 paper] 在 Harness 落地？"

**应对**：T-6 把 Anthropic 三篇 harness blog + SWE-Bench Pro paper + 1~2 篇近期 agent paper（如 Voyager / OS Atlas / Lab Skills）啃熟。

### 暗线 3：你"在没用过的语言/框架下"能不能 ship
JD 第 3 条："**没有直接经验的领域**有质量保证的编程"——这是**学习能力**的硬测试。

**面试场景预测**：可能现场给你一个你没用过的（如 Bun bundler / Tauri / Rust async）让你 10 分钟做个 mini task。考的不是技术细节，是**你怎么用 AI 辅助快速搜索 / 验证 / 写代码**的工作流。

**应对**：T-5 ~ T-3 用 Cursor 在一个你不熟的栈下做一个 100 行的 demo（如：用 Go 写 MCP server，或用 Rust 写 sandbox wrapper）。

### 暗线 4：你 ship 不 ship
和 PM 版的暗线一样。**带能跑的东西去面试**：
- 一个 minimal harness GitHub repo
- 一篇关于 Anthropic harness 的中文技术博客
- 一个开源 Agent 工具的 PR（哪怕只是 docs / 小 bug fix）

### 暗线 5：你是不是长期主义者
DeepSeek 不发奖金不赛马，靠认同感留人。会问：
- 为什么不去字节腾讯（钱更多）/ Cursor SF（更国际化）？
- 你看 5 年后的 DeepSeek 长什么样？

**准备答案**："我看好 DeepSeek 把模型 + Harness 做成中国 research-driven product 的能力——在国内能像 Anthropic 一样做 reference harness 的团队只有这一个。"

## 八、一句应试钥匙

> **每个答案问自己一句："这个回答能不能体现『我会写』+『我有品味』+『我能和研究员对话』三件事？"**
