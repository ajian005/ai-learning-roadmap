# 11 · 面试后复盘模板与跟进话术

> 面试结束的第 1 分钟～第 7 天，是大多数人会"放空 / 焦虑 / 错过"的窗口。这一份是结构化复盘 + 高质量跟进的执行手册。
>
> **核心信念**：复盘比面试本身更增长——面试是单次的，复盘的能力会陪伴你下一次面试和你做 PM 的每一天。

## 二、感谢邮件模板（面试当天发，最晚次日早晨）

### 模板 1·HR 收件人

```
主题：Re: DeepSeek Agent Harness PM 面试 - 感谢

[HR 老师好]，

非常感谢您安排今天和 [面试官姓名] 的交流——这是我近期聊得最有质量的一次产品对话。

谈话中提到的几个点我会继续思考：
1. [面试官提到的关于团队/产品的某个具体观点]
2. [你聊到但回家想得更清楚的某个题]

我把今天聊到的内容更新进了我面试前那份《DeepSeek Code Harness V0.1 产品提案》——附件是 V0.2，你们看着方便可以转给团队，也欢迎任何反馈。

如果还有后续轮次，我很期待和团队继续聊。

[你的名字]
[你的微信/Twitter/GitHub]
```

### 模板 2·直接给面试官（如果 HR 给了 cc）

```
主题：Re: 今天面试 - 几个延伸思考

[姓名/老师好]，

谢谢您今天的时间，特别是您在 [具体话题] 上的判断让我回家又想了一晚。

简短分享两个回家后想清楚的：
1. 关于您问的 [题目]：我面试时答 [答案]，回家后觉得更准确的角度是 [新观点]——主要是因为 [原因]
2. 关于您聊到的 [团队/产品观点]：我去看了 [资料]，进一步想到 [洞察]

附件是我的提案 V0.2，把今天聊到的几个点合进去了。如果有时间欢迎指正。

期待保持联系。

[你的名字]
```

### 邮件写作 5 条原则

1. **24 小时内发**——超过 48 小时收益急剧降低
2. **不超过 200 字**——太长是骚扰
3. **必须有 1 个"具体延伸思考"**——否则就是套话邮件
4. **附件用 PDF 不用 docx**——可读性 + 防止修改
5. **抄送 HR**——保持透明，方便流程

## 三、被推进下一轮 / 等待中的跟进节奏

### 等待 1~3 天
- 完全不动作（HR 这边还在内部 calibrate）

### 等待第 4 天
- 给 HR 发简短跟进：
> "[HR 老师好]，简单跟进一下——上周二的面试结束后，请问当前流程进展是什么阶段？如果团队需要补充材料（代码 demo / 提案细化），随时告诉我。"

### 等待 7~10 天 没消息
- 礼貌但坚定地再 ping 一次：
> "您好，再次跟进上次面试。理解团队招聘节奏紧；如果方便能否告知大致 timeline 或方向？我也好做时间安排。任何结果我都尊重。"

### 等待 14 天 没消息
- 接受为"软拒"。但**不要写'怪罪'信**。可以发一封"建立长期关系"信：
> "[HR 老师好]，看来这次时机不合适——完全理解，团队招聘标准高。我会继续关注 Harness 团队的进展，如果未来有适合的机会，希望能再次被考虑。如果方便分享一些反馈给我，会非常感激——我把它当成成长信号。"

## 四、被拒后的 3 种二次转化话术

### 转化 1·要 specific feedback

```
谢谢告知。完全尊重决定——但能不能请您（或团队）分享 1 句最 specific 的 feedback？
比如"在 X 题表现弱"或"对 Y 主题不够熟"。这对我下一段成长意义很大。
```

> 大约 30~50% 的 HR 会回。回的内容比 offer 本身更有价值。

### 转化 2·留长期关系

```
完全理解。我会继续关注 DeepSeek Harness 的进展——
如果未来 6~12 个月团队再有合适的 role，希望可以再次被考虑。
我也会持续在 [博客/社区] 写 Harness 相关内容，欢迎你们看到任何想 challenge 的地方。
```

> 1~2 年后这种关系经常变成 offer——而且是更适合的 offer。

### 转化 3·要"我哪里可以变得更强"

```
我有一个不算请求的请求——
如果让您给我 1 个建议，"下次我面什么角色 / 公司能 raise 我的 hit rate"，您会怎么说？
我不指望命中率，但您看了我这场面试后的判断，对我比任何模板有用。
```

> 这种"求成长信号"的姿态让 HR 印象极深——多年后如果他/她跳槽，可能反过来找你。

## 五、被录用后的"还在选公司"话术

> 假设你拿到 DeepSeek offer，同时也在和其他公司谈：

```
非常感谢这份 offer。能拿到我感到很荣幸。

实话和您说我目前还有 [N] 个在 final stage 的 process，我想在 [日期] 之前给您最终答复。
原因是这个决定对我是 3~5 年级的，我希望对所有给我机会的团队都 fair。

如果团队这边有 deadline 我能尽量配合，但希望能争取到 [合理时间]。
```

> 透明 + 给具体 deadline + 尊重对方——这是高级 PM 的 signature。

## 七、心理建设：被拒不等于"我不够好"

| 拒绝原因 | 真实概率 |
| --- | --- |
| 你"不够好" | 20% |
| 团队已有"更合适"的人 | 35% |
| 业务方向变了 / hiring freeze | 15% |
| 文化不匹配（不是好坏，是 fit） | 20% |
| 你太"贵" / 太"便宜" | 10% |

> 80% 的拒绝 **不是你能力问题**——是各种你不知道也无法控制的变量。
>
> 而 100% 的拒绝你都能从复盘里榨出可迁移经验。

---

## 八、一句应试钥匙

> **复盘的最高价值不是"为了下次面赢"，是"看清自己作为 PM 在 2026 年的真实段位"。**

> 段位看清了，下一步要补什么、要练什么、要去哪种环境都有答案。

> 面试是镜子，复盘是显影液。
